Descobreix Espanya







La API que he triat va sobre el clima de cada poblacio en temps real. Perque es mostri el clima de la poblacio triada, has de clicar sobre una imatge de les que et mostra, llavors directament es mostra el recuadre de el clima(temperatura). He tret la API de la pagina OpenWeather. Perque la API funciones he creat una clau que li he posat com a nom clima. Un cop creada et donava una clau amb molts digits, un cop amb aquells digits els he introduit al js perque pugues funcionar la API de clima. He triat la API de temperatura en temps real perque hem semblava interesant i curios i m'ha fet gracia introduir aquesta API al meu projecte.

He triat una altre API on et mostra la bateria actual del teu dispoitiu. Et mostra el nivell de bateria, si esta cargant o no i el minuts que li falten perque s'acabi la beteria. He volgut adjuntar aquesta API perque tambe m'ha semblat interseant que te mostri el nivell de bateria i el temps que li falta perque s'acabi. Aquesta Api le tret del link del document que ens van pasar, es diu BatteryManager.
